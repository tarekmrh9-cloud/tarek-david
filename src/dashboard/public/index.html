📖 دليل النشر على Railway

      

        

          
1

          

            
أنشئ حساباً على Railway

            
اذهب إلى railway.app وسجّل دخول بـ GitHub

          

        

        

          
2

          

            
ارفع الكود على GitHub أولاً

            
استخدم قسم "رفع على GitHub" أعلاه

          

        

        

          
3

          

            
New Project → Deploy from GitHub Repo

            
اختر الريبو castrolmocro/divid-apk

          

        

        

          
4

          

            
أضف متغيرات البيئة (Environment Variables)

            
في لوحة Railway → Variables:

            
NODE_ENV=production


            
PORT=3000


            
DASHBOARD_PASSWORD=david2025

          

        

        

          
5

          

            
أدخل رابط Railway في تطبيق Android

            
من تطبيق DAVID → الإعدادات → عنوان السيرفر

            
https://your-project.railway.app

          

        

      

    


    
    

      
📱 الهاتف كسيرفر (بدون انترنت خارجي)

      

        شغّل البوت مباشرة من هاتفك الأندرويد — البوت والتطبيق يعملان على نفس الجهاز.
      


      

        

          

          📲 إعداد الهاتف كسيرفر
        

        
# 1. ثبّت Termux من F-Droid (لا من Play Store)
pkg update && pkg upgrade -y
pkg install nodejs git -y
git clone https://github.com/castrolmocro/divid-apk
cd divid-apk && npm install
node index.js
✅ البوت يعمل على http://localhost:5000
# 2. افتح التطبيق وسيتصل تلقائياً بـ localhost
